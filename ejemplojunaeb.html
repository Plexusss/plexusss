Soporte del Servicio: Aplicación

Escala de Habitos UX: Bueno

Descripción de experiencia de usuario con el servicio: Esta aplicación se utiliza para realizar comrpas de supermercados. Al minuto de realizar la compra, facilita un código dinámico o un código QR el cual tiene un tiempo de expiración, se considera una buena experiencia de usuario ya que da a conocer de inmediato dos códigos si es que uno no funciona.
